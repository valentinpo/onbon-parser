Инструкция для «Чайников» 🫖
Всё просто! Следуй шагам по порядку.
📍 Подготовка (сделать 1 раз)
Убедись, что все файлы в одной папке
Например: C:\Users\pw\Desktop\парсер онбон\
Там должны лежать:
✅ onbon_parser.py
✅ split_by_model.py
✅ organize_files.py
✅ onbon_slugs.txt (твой список из 131 модели)
✅ requirements.txt
Открой консоль в этой папке
Зажми Shift → кликни правой кнопкой мыши в пустом месте папки → выбери «Открыть окно команд» или «Open in Terminal».
Или открой cmd, напиши cd и перетащи папку мышкой в окно, нажми Enter.
🚀 Запуск (по шагам)
Шаг 1. Установи библиотеки
(Делается один раз при первом запуске)

bash
1 py -m pip install -r requirements.txt


✅ Успех: Появится надпись Successfully installed...
Шаг 2. Скачай характеристики (Парсинг)
(Самый важный шаг! Занимает 2-3 минуты)

bash
1 py onbon_parser.py


✅ Успех: Появится файл onbon_catalog.csv и надпись ✅ Готово: 128 успешно...
⏳ Жди: Не закрывай окно, пока не дойдёт до 131 модели.
Шаг 3. Разбей на файлы по моделям
(Делает отдельный файл для каждой модели)

bash
1 py split_by_model.py

✅ Успех: Появится папка products_by_model, внутри много CSV-файлов.
🧹 Бонус: Скрипт сам удалит дубликаты.
Шаг 4. Разложи по папкам
(Создаёт отдельную папку для каждой модели)

bash
1 py organize_files.py

✅ Успех: Внутри products_by_model появятся папки ovp-m1, ovp-m2 и т.д.
📂 Итог: В каждой папке лежит файл с характеристиками именно этой модели.
📁 Где результат?
Общий файл: onbon_catalog.csv (можно открыть в Excel)
По моделям: Папка products_by_model → выбери нужную модель → открой CSV.

❌ Что-то пошло не так?
Ошибка
Решение
No module named 'requests'
Выполни Шаг 1 ещё раз
File not found
Проверь, что консоль открыта в папке проекта
404 Not Found в логе
Некоторые модели устарели, это нормально (парсер пропустит их)
Всё на английском
В файлах CSV русские буквы должны отображаться корректно (Excel может спросить кодировку — выбирай UTF-8)
🔄 Как обновить данные позже?
Просто запусти Шаг 2, Шаг 3 и Шаг 4 заново.
Скрипты автоматически перезапишут старые данные новыми.
Готово! 🎉
Теперь у тебя есть полная база характеристик всех 131 моделей Onbon.
